# 🌊 Deep Sea Maritime - Frontend

This is the Next.js 16 frontend for the [Deep Sea Maritime](..) company profile.

## 🚀 Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:3000`.

### Building for Production

```bash
npm run build
npm start
```

## 🛠️ Tech Stack

- **Next.js 16** (React 19)
- **Tailwind CSS 4**
- **Lucide React** (icons)
- **Axios** (API requests)

---

## 🎨 Design System

The frontend follows the **Deep Sea** design philosophy:
- **Primary Colors**: Deep oceanic blues and teals.
- **Components**: Wave-based transitions, "porthole" cards, and hull-shaped sections.
- **Animations**: Subtle water-like micro-animations for interactions.
